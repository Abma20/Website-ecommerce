Aplikasi web sederhana untuk toko online parfum. Sistem ini memungkinkan pengunjung untuk melihat katalog parfum, melakukan pemesanan, dan memungkinkan admin untuk mengelola produk.

Dibangun menggunakan PHP Native dan MySQL, proyek ini cocok untuk pemula yang ingin mempelajari alur dasar E-Commerce (CRUD, Session, Upload File, dan Transaksi Sederhana).

Dashboard admin (admin_produk.php).
